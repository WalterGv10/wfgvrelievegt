# RelieveGt - Web

Plataforma web para **RelieveGt**, especialistas en impresión 3D desde llaveros personalizados hasta prototipado industrial.

## Tecnologías

- **Core**: React 19 + Vite
- **Estilos**: Tailwind CSS v4
- **Efectos**: OGL (Fondo Iridiscente)
- **Routing**: React Router DOM
- **SEO**: React Helmet Async

## Instalación y Uso

1. **Instalar dependencias**:
   ```bash
   npm install
   ```

2. **Iniciar servidor de desarrollo**:
   ```bash
   npm run dev
   ```

3. **Construir para producción**:
   ```bash
   npm run build
   ```

## Estructura del Proyecto

- `src/components`: Componentes reutilizables (UI, Efectos).
- `src/pages`: Vistas de la aplicación.
- `src/hooks`: Lógica reutilizable.
- `src/layouts`: Estructuras de página.

## Características

- Diseño **Mobile-First**.
- Optimización SEO integrada.
- Fondo interactivo con WebGL (Iridescence).
